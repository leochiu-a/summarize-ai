隱私權政策

Summarize AI Buddy Chrome Extension ・ 最後更新：2026-07-23

Summarize AI Buddy（以下稱「本擴充功能」）是一個在網頁右下角顯示互動角色、 協助你摘要目前瀏覽頁面內容的 Chrome 擴充功能。我們非常重視你的隱私，本頁說明 本擴充功能會存取哪些資料、資料如何被處理，以及資料會不會被傳送出去。

一句話總結

所有摘要都在你的裝置本機完成，網頁內容不會被傳送到我們或任何第三方的伺服器。 我們沒有後端伺服器，也不會蒐集、分析或出售你的任何資料。

本擴充功能會存取什麼

  • 目前分頁的網頁內容：當你點擊小夥伴時，本擴充功能會讀取當前頁面 可見的文字內容（或用 Readability 演算法抽取的文章正文），做為產生摘要的輸入。

摘要如何產生

本擴充功能使用 Chrome 瀏覽器內建的 Summarizer API （由 Gemini Nano 模型驅動）。這個模型直接在你的裝置上運算， 頁面內容與產生的摘要全程只存在於你的瀏覽器裡，不會經過網路傳送到任何伺服器 ——包含我們自己的伺服器在內，因為我們根本沒有經營任何伺服器。

本機儲存了什麼

為了讓體驗更好，本擴充功能會用 Chrome 的 storage.local API 把以下資料存在你自己的裝置上：

  • 使用者設定：你在設定頁選擇的語氣、摘要類型、是否開啟「每頁自動摘要」。
  • 摘要快取：同一個網址在 30 分鐘內重新開啟時，直接使用先前產生的摘要結果，避免重複運算。快取內容包含頁面網址與產生的摘要文字。

這些資料只存在你自己的瀏覽器設定檔中，會跟著 Chrome 帳號同步（如果你有開啟 Chrome 同步）， 但不會被傳送給我們或任何第三方。移除本擴充功能時，這些資料會一併被 Chrome 清除。

我們不會做的事

  • 不會蒐集或傳送你的瀏覽紀錄、頁面內容或個人資料到任何伺服器
  • 不使用任何分析、追蹤或廣告相關的第三方服務（沒有 Google Analytics、沒有像素、沒有 SDK）
  • 不會將資料出售或分享給第三方——因為我們根本不會傳輸或蒐集任何資料
  • 不會在背景存取與摘要功能無關的頁面資料

為什麼需要「讀取所有網站資料」的權限

本擴充功能需要在你造訪的網頁上顯示互動角色、並在你主動點擊時讀取當前頁面內容， 所以 Chrome 會將此列為「可讀取所有網站的資料」。這個權限只用來： (1) 在頁面右下角顯示小夥伴角色的介面、(2) 在你點擊摘要時擷取當前頁面的文字內容。 除此之外不會做任何其他用途，也不會在你未點擊摘要時讀取或處理頁面內容。

圖示與素材

介面中使用的動畫表情符號來自 Google Noto Emoji， 已隨本擴充功能打包在本機，執行時不會向任何外部網址發出請求載入圖片。

政策異動

若本隱私權政策有重大異動，我們會更新本頁面頂端的「最後更新」日期。建議在更新 本擴充功能版本後，回來確認本頁面內容。
